## CO1 Reflection
This assessment required applying cloud computing fundamentals — service models, deployment
models, virtualization, web services, and elasticity — to a concrete project rather than
describing them in the abstract. Mapping every Campus Connect module to a specific cloud
service, and justifying why, was what turned textbook definitions of IaaS/PaaS/SaaS/FaaS into
an applied design skill (CO1, BL2).
